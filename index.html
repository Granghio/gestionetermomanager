Salvataggio Esterno Automatico

Usa questa funzione per salvare i dati in un file sul tuo computer in tempo reale.

Come funziona:
1. Clicca il tasto verde qui sotto.
2. Scegli dove salvare il file (es. sul Desktop e chiamarlo clienti_db.json).
3. Ogni modifica aggiornerà quel file in automatico.
Nessun file collegato.

Backup Manuale (Classico)

Come funziona:
1. Clicca su "Scarica Backup" per salvare un file JSON sul tuo computer.
2. Puoi conservare il file dove preferisci (es. cloud, chiavetta).
3. Per ripristinare, clicca su "Ripristina Backup" e seleziona il file salvato.
Consiglio: esegui un backup manuale dopo ogni modifica importante.
